\doxysubsection{Подробное описание}
Класс для генерации RSA-\/ключей 

Класс \doxylink{class_r_s_a_key_generator}{RSAKey\+Generator} предоставляет статические методы для генерации RSA-\/ключей и их преобразования в текстовый формат PEM. Это позволяет использовать функциональность без необходимости создания экземпляра класса.
