\doxysubsection{Подробное описание}
Класс для криптографических операций с использованием Open\+SSL. 

Предоставляет методы для\+:
\begin{DoxyItemize}
\item вычисления SHA-\/256 хеша,
\item подписи данных с помощью RSA приватного ключа,
\item проверки подписи с помощью RSA публичного ключа.
\end{DoxyItemize}

\begin{DoxyWarning}{Предупреждения}
Для работы требуется библиотека Open\+SSL. 
\end{DoxyWarning}
