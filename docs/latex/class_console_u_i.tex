\doxysubsection{Подробное описание}
Класс для работы с консольным интерфейсом системы 

Предоставляет методы для форматированного вывода информации\+:
\begin{DoxyItemize}
\item Заголовки и баннеры
\item Цветовые стили для разных типов сообщений
\item Элементы пользовательского интерфейса 
\end{DoxyItemize}
