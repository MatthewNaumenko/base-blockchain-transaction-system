\doxysubsection{Подробное описание}
Класс для управления сохранением и шифрованием данных. 

Предоставляет статические методы для шифрования данных, которые могут быть использованы для безопасного хранения информации.

Шифрует строку с использованием заданного ключа. 

Использует алгоритм AES-\/256-\/\+CBC для шифрования данных. Вектор инициализации (IV) генерируется случайным образом и добавляется к результату.


\begin{DoxyParams}{Аргументы}
{\em plaintext} & Исходная строка для шифрования. \\
\hline
{\em key} & Ключ шифрования (должен быть длиной 32 байта для AES-\/256). \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Возвращает}
Зашифрованная строка, содержащая IV и зашифрованные данные. 
\end{DoxyReturn}
