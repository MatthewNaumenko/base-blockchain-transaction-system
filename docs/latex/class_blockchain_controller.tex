\doxysubsection{Подробное описание}
Класс-\/посредник для управления блокчейном. 

Класс-\/посредник \doxylink{class_blockchain_controller}{Blockchain\+Controller} управляет блокчейном и предоставляет методы для обработки транзакций, проверки валидности блокчейна, регистрации пользователей и работы с балансами.
