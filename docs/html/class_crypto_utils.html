Подробное описание

Класс для криптографических операций с использованием OpenSSL.

Предоставляет методы для:

  • вычисления SHA-256 хеша,
  • подписи данных с помощью RSA приватного ключа,
  • проверки подписи с помощью RSA публичного ключа.
Предупреждения
Для работы требуется библиотека OpenSSL.
