Подробное описание

Класс для управления RSA-ключами пользователей.

Класс KeyManager предназначен для управления RSA-ключами пользователей. Он позволяет генерировать, сохранять и предоставлять доступ к публичным ключам Приватный ключ сохраняется в формате PEM и удаляется из памяти.
