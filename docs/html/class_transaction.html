Подробное описание

Класс, представляющий криптографически защищенную транзакцию блокчейна

Инкапсулирует логику валидации входных данных, генерации цифровой подписи и сериализации транзакции. Все операции выполняют предварительные проверки целостности данных.

double value,

Создает транзакцию с базовой валидацией полей

Аргументы
fromАдрес отправителя (только валидные Base58Check-адреса)
toАдрес получателя (аналогично from)
valueПоложительная сумма перевода (<=0 инициализирует 0)
metaМетаданные (обрезаются до 512 символов, могут быть пустыми)
Исключения
implicitнеявная инициализация невалидных полей пустыми значениями/нулем
Заметки
Автоматически генерирует txId и timestamp через CryptoUtils и TimeUtils
